# ETF Picker 📊

네이버 금융 실시간 데이터 기반으로 조건에 맞는 ETF를 찾아주는 웹 프로그램입니다.

## 기능

- **1,000개 이상** 국내 상장 ETF 검색
- 다양한 필터 조건: 국내/해외, 자산유형, 레버리지/인버스, 운용사, 추종지수, 순자산, 수익률, 등락률
- 실시간 네이버 금융 데이터 연동
- 검색 결과 클릭 시 네이버 금융 상세 페이지 연결

## 설치

```bash
python -m pip install flask requests beautifulsoup4 lxml flask-cors
```

## 실행

```bash
python app.py
```

브라우저에서 `http://localhost:5000` 접속

또는 `start.bat` 파일을 더블클릭

## 데이터 소스

- **네이버 금융** (https://finance.naver.com): 전체 ETF 시세, 수익률, 시가총액
- **네이버 금융 개별 페이지**: 추종지수, 운용사, 기간별 수익률 (백그라운드 수집)

> 첫 실행 시 상세 정보 수집에 약 1~3분이 소요됩니다.
> 수집된 데이터는 30분간 캐시됩니다.

## 필터 항목

| 필터 | 설명 |
|------|------|
| 투자 지역 | 국내 / 해외 |
| 자산 유형 | 주식 / 채권 / 원자재 / 부동산 / 혼합 / 머니마켓 |
| 상품 유형 | 일반 / 레버리지 / 인버스 |
| 상품명 | ETF 이름 키워드 검색 |
| 추종지수 | 추종 지수 키워드 검색 |
| 운용사 | 삼성자산운용, 미래에셋자산운용 등 |
| 순자산 규모 | 최소 순자산(억원) 설정 |
| 3개월 수익률 | 범위 설정 |
| 당일 등락률 | 범위 설정 |

## 면책 조항

이 프로그램은 투자 권유를 목적으로 하지 않습니다.
과거 수익률이 미래 수익률을 보장하지 않습니다.
